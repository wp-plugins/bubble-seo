== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Bubble SEO'
3. Activate 'Bubble SEO' from your Plugins page.
4. Visit 'Setting > 'Bubble SEO'.
